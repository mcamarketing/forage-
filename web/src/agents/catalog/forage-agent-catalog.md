## IMPLEMENTATION

Each agent follows this pattern:

```typescript
const agent = {
  name: "forage-[role]",
  description: "...",
  foragerTools: [...],
  knowledgeGraph: {
    claims: true,
    signals: true,
    regime: true
  },
  systemPrompt: `You are...`,
  tools: [/* LangChain tools */]
}
```

Deploy via `/api/sales-agent` with `action` parameter or create individual endpoints.
